# nexfilesd

A build-automation agent customized for [nexfiles.kjonigsen.net](http://nexfiles.kjonigsen.net).

## Installation

Clone repo. Customize builds in src/nexfilesd/core.clj.

## Usage

    $ lein run

## Options

No options are required nor parsed.
